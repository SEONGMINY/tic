# tic — 사용자 흐름

## F1: 첫 실행
```
앱 실행
 → 시스템: 캘린더 권한 요청 팝업
 → 시스템: 미리 알림 권한 요청 팝업
 → 월간 뷰 (모든 캘린더 기본 활성화)
```
거부 시: 빈 캘린더로 앱 동작, 재시도/차단 없음.

## F2: 월간 뷰 (기본 화면)
```
[월간 뷰]
 내비: "2026년" ⚙️ 🔍 +
 그리드: 일정 있는 날 오렌지 점, 오늘 = 오렌지 원
 
 날짜 탭 → F3 (해당 날짜 일간 뷰)
 Pinch out → F7 (년간 뷰)
 "2026년" 탭 → 오늘로 이동
 + 탭 → F5 (일정 생성, 빈 폼)
 🔍 탭 → F6 (검색)
 ⚙️ 탭 → F8 (설정)
```

## F3: 일간 뷰
```
[일간 뷰]
 내비: "← 4월" ⚙️ 🔍 +

 오늘인 경우:
   [다음 행동 카드]
    - 다음 캘린더 이벤트 또는 시간 있는 리마인더 1개 표시
    - 리마인더: 체크 완료 가능
    - 캘린더 이벤트: 표시만
    - 남은 일정 없음: 카드 숨김

 [종일/다중일 이벤트 영역] (타임라인 상단)

 [24시간 타임라인 - Apple Calendar 스타일]
  - 이벤트 블록: 시간 기준 배치, 캘린더 고유 색상
  - 표시 대상: 캘린더 이벤트 + 시간 있는 리마인더만
  - 겹치는 일정: column-packing (나란히 배치)
  
 [📋 FAB - 우하단]
  - 탭 → 시간 없는 리마인더 체크리스트 (bottom sheet)

 제스처:
  좌우 스와이프 → 이전/다음 날
  Pinch out → 월간 뷰
  타임라인 시간대 꾹 → F5 (일정 생성, 시간 자동 입력)
  이벤트 블록 탭 → F4 (일정 수정)
  이벤트 블록 꾹 → Context menu (수정/삭제/완료)
```
과거 날짜: 다음 행동 카드 숨김, 타임라인만 표시.

## F4: 일정 상세 / 수정
```
[Bottom Sheet - 수정 모드]
 F5와 동일한 폼
 기존 데이터 자동 입력
 하단: 빨간 "삭제" 버튼 → 확인 alert → EventKit에서 삭제
```

## F5: 일정 생성
```
[Bottom Sheet - 생성 모드]
 진입점:
  a) + 버튼 → 모든 필드 비어있음
  b) 타임라인 꾹 → 시작 시간 = 눌린 시간, 종료 = +1시간

 폼:
  제목 [필수]
  설명
  저장 위치: [드롭다운 - 모든 캘린더 + 리마인더 리스트, 플랫, 아이콘 구분]
    → 캘린더 선택: 시작/종료 필수 (기본 09:00-10:00)
    → 리마인더 선택: 시간 선택/미선택 가능
  반복: [없음|매일|매주|2주마다|매월|매년]
  알림: [없음|5분전|15분전|30분전(기본)|1시간전]
  
 저장 버튼: 필수 필드 비어있으면 비활성화
 
 저장 시:
  → EventKitService가 Apple Calendar/Reminders에 쓰기
  → 로컬 알림 스케줄 등록
  → 첫 일정인 경우: 알림 권한 요청
```

## F6: 검색
```
[Navigation Push]
 검색바 (자동 포커스)
 
 빈 입력:
  최근 검색 기록 리스트
  각 항목: 검색어 + ✕ 삭제 버튼
  
 입력 시:
  실시간 필터링 결과
  날짜별 그룹핑 (섹션 헤더)
  범위: 과거 + 미래 전체
  
 결과 탭 → F4 (일정 수정)
 검색 제출 → SearchHistory에 저장 (SwiftData)
```

## F7: 년간 뷰
```
[년간 뷰]
 12개월 미니 캘린더 그리드
 일정 있는 날에 점 표시
 
 월 탭 → F2 (해당 월의 월간 뷰)
 Pinch in → F2 (월간 뷰)
```

## F8: 설정
```
[Bottom Sheet]
 모든 캘린더 + 리마인더 리스트 나열
 각 항목: 켜기/끄기 토글
 기본값: 전체 활성화
```

## F9: Live Activity
```
트리거 A (포그라운드): 일정 30분 전 → Live Activity 자동 시작
트리거 B (백그라운드): 로컬 알림 발송 → 사용자 탭 → 앱 열림 → Live Activity 시작

[잠금화면 / Dynamic Island]
 일정 제목 + 시간 범위
 [완료] [10분 후 알림]
 
 완료 → 완료 처리 (리마인더) 또는 해제 (이벤트) + 활동 종료
 스누즈 → 10분 후 알림 예약 + 활동 유지
 일정 종료 시간 도달 → 활동 자동 종료
 
 다음 일정이 30분 이내? → 즉시 새 활동 시작
```

## F10: 위젯 인터랙션
```
[Small 위젯]
 다음 일정 1개: 제목 + 시간
 탭 → 해당 날짜 일간 뷰로 앱 열기

[Medium 위젯]
 다음 3-4개 일정, 초과 시 "+N개 더"
 탭 → 해당 날짜 일간 뷰로 앱 열기

둘 다: App Intents를 통한 완료/스누즈 버튼 (Interactive Widget)
```

## F11: 알림 액션
```
[로컬 알림 - 일정 시간]
 제목: 일정 제목
 액션:
  [완료] → 리마인더 완료 처리 / 이벤트 해제
  [10분 후] → 스누즈 알림 예약
```
